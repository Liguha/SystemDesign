## Развертывание системы
```bash
docker compose up --build -d
```

После развёртывания к контейенерам можно получить доступ по:
- API - `localhost:8080`
- PostgreSQL - `localhost:5432`.

**Переинициализация БД:**
```bash
docker compose down -v
docker compose up --build -d
```
